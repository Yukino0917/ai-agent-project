# 🤖 智能问答 Agent

基于大模型的智能问答系统，支持知识库检索、数学计算和联网搜索。

## 功能特性
- 知识库检索：基于 ChromaDB 向量数据库
- 数学计算：支持复杂数学表达式
- 联网搜索：获取最新信息
- 多轮对话：支持上下文记忆
- Web 界面：基于 Streamlit

## 技术栈
- Python 3.12
- OpenAI API（兼容其他 LLM API）
- ChromaDB（向量数据库）
- Streamlit（Web 界面）

## 快速开始
1. pip install -r requirements.txt
2. 设置环境变量 OPENAI_API_KEY
3. streamlit run main.py
